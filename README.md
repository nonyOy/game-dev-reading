# Game Dev Reading

每天阅读游戏制作相关内容，并记录新的想法、方法和可实践的经验。

## 记录方式

- `notes/`：按日期保存每日阅读笔记。
- `templates/daily-note.md`：每日笔记模板。
- `topics.md`：按主题沉淀长期索引。

## 建议命名

每日笔记可以使用：

```text
notes/YYYY-MM-DD.md
```

例如：

```text
notes/2026-05-28.md
```

- 关卡设计
- 系统设计

- 叙事设计
