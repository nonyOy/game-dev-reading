# 2026-05-28：读《杀戮尖塔》拆解，想想它为什么成功

今天读了三篇关于《杀戮尖塔》的文章，又看了一张系统脑图。三篇文章的侧重点不太一样：GameRes 那篇更像是从游戏策划角度讲怎么拆解游戏，杀戮尖塔攻略站那篇更强调动态构筑、资源转化和数值价值，知乎那篇则更像一份系统总览，把战斗、角色、地图、遗物、药水、生命周期和社区生态都铺开了。

读完以后，我不太想只用“卡牌 + Roguelike”来概括它。这个说法当然没错，但太像标签了。它只能说明《杀戮尖塔》长得像什么，不能解释为什么它能一直让人想再开一局。

我现在更愿意这样理解它：《杀戮尖塔》的成功，在于它把很多看似分散的系统，都压进了同一个风险收益问题里。玩家每做一个选择，其实都在问：

我现在付出什么，能换来之后更高的胜率吗？

这句话几乎能套到游戏里的所有核心选择上。拿牌是在用牌库厚度换潜在强度；打精英是在用血量风险换遗物；火堆升级是在用当前安全感换长期强度；商店删牌是在用金币换稳定性；事件是在用不确定性赌收益；药水是在用一次性资源换一场战斗的通过率。

所以它不是单纯系统多，而是系统之间互相咬合。玩家不是在处理一堆并列模块，而是在不断计算同一套账。

## 一、它的目标简单，但过程一直在变化

《杀戮尖塔》正式版是在 2019 年发售的，由 Mega Crit Games 开发。游戏目标表面上非常简单：选择一个角色，从塔底一路往上走，打过三个楼层的 Boss。如果玩家收集三把钥匙，还可以进入第四幕挑战心脏。

这个目标并不复杂，也没有很重的剧情包装。玩家一眼就能明白自己要做什么：活下去，变强，爬塔。

但它真正好玩的地方不在目标，而在抵达目标的过程每次都会变化。地图路径、敌人组合、卡牌奖励、商店内容、事件结果、遗物顺序、药水掉落，这些东西都会变。玩家不能照搬上一局的答案，因为这一局给你的材料不一样。

这也是 Roguelike 结构给它带来的基础可重玩性。死亡以后重开，每局资源重新积累，角色和部分卡牌、遗物通过游玩逐步解锁。这个框架保证了“再开一局”的动力。

但我觉得更关键的是，《杀戮尖塔》的随机性不是为了让玩家认命，而是为了让玩家重新判断。很多游戏也随机，但随机之后玩家没有太多补救空间，就容易变成“这把运气不好”。《杀戮尖塔》不是这样。它会让你遇到不确定性，但也给你足够多的信息和工具，让你有机会通过路线、拿牌、删牌、药水和出牌顺序去调整。

所以它的随机性更像是在不断打乱你的计划，而不是直接宣判你的失败。

## 二、动态构筑才是游戏真正的骨架

传统卡牌游戏通常是先组好牌，再进入战斗。构筑和战斗是两个阶段。玩家在战斗前规划卡组，在战斗中验证卡组。

《杀戮尖塔》不一样。它把构筑放进了爬塔过程里。

开局时，玩家手里只有一副很普通的初始牌组。后面每场战斗结束后的卡牌奖励、商店里的购买与删牌、事件里的选择、火堆升级、遗物效果，都会让牌组一点点变形。你不是带着一套完成品进塔，而是在塔里边打边做牌组。

这就让构筑从“准备工作”变成了“游戏本体”。

我觉得这点非常重要。因为它让玩家一直处在一个半成型状态里。你的牌组永远不是完全理想的，总有缺口：可能缺输出，可能缺防御，可能缺过牌，可能启动慢，可能太厚，可能有诅咒，可能拿到了核心但没有配套组件。

玩家每次拿牌都要想两个方向：

这张牌能不能解决我现在的生存问题？  
这张牌会不会影响我后面的成型方向？

前期一张普通攻击牌可能很重要，因为没有输出就打不过精英；但后期同一张普通攻击牌可能是污染，因为它会降低抽到核心牌的概率。这个动态价值变化很迷人。

如果一个游戏只问“这张牌强不强”，那决策会比较浅。  
《杀戮尖塔》问的是：“这张牌在我这一局、这个阶段、这个牌组里值不值得拿？”

这就是它构筑深度的来源。

## 三、战斗循环很简单，但每一步都有机会成本

战斗的基础循环其实很好懂：每回合抽 5 张牌，获得 3 点能量，使用手牌，结束回合后弃掉手牌，敌人行动，然后进入下一回合。

这个流程简单到新手很快就能理解。但越玩越会发现，真正的难点不在“规则看不看得懂”，而在“有限资源怎么分配”。

能量是最直接的限制。玩家一回合想做的事情很多：打伤害、防御、过牌、启动能力、挂易伤、使用功能牌。但能量只有那么几点。设计师通过不同费用的牌制造了很多取舍：高费牌效果强但笨重，低费牌灵活但单张价值可能低，0 费牌不耗能量但会占抽牌位。

所以游戏里没有真正免费的东西。即使是 0 费牌，也会占据一张手牌位置，也会影响你抽到其他牌。

手牌和牌库循环也很关键。用过的牌进入弃牌堆，抽牌堆空了以后重新洗牌。牌组越精简，关键牌出现频率越高；牌组越臃肿，强牌就越不稳定。初始的打击、防御，后面被敌人塞进来的状态牌，事件给的诅咒，都可能让牌库变差。

这也是为什么“跳过卡牌奖励”在这个游戏里是一个严肃选择。很多游戏给奖励就一定拿，但《杀戮尖塔》里，不拿有时候才是对的。因为你不是在收集越多越好，而是在维护一个可以稳定运转的系统。

我觉得这点很高级。它让“奖励”本身也变成风险。

## 四、格挡清零和敌人意图，让战斗变成明牌解题

《杀戮尖塔》的格挡在回合结束时会清零，除非有特殊能力，比如铁甲战士的“壁垒”。这个规则看起来只是一个小设定，但影响非常大。

如果格挡能一直保留，玩家很容易进入龟缩节奏，先攒一堆防御再慢慢输出。但格挡清零以后，防御变成了即时响应。敌人这一回合要打你，你这回合才需要防；敌人不攻击，你这回合打防御就可能浪费。

这让每一回合都要重新做攻防判断。

敌人意图系统又把这个判断变得很公平。敌人下一步要攻击、加防、上状态、召唤，游戏都会提前告诉你。玩家不是在猜敌人，而是在处理一个明牌局面。

我觉得这就是《杀戮尖塔》战斗体验很好的原因之一。它把挑战从“信息猜测”转移到了“决策质量”。玩家知道敌人要做什么，但不一定有足够资源完美处理。你知道敌人下回合要打 18，不代表你一定能防住；你能防住，也不代表防住就是最优，因为你可能错过斩杀窗口。

这种失败很容易复盘。玩家输了不会只觉得“我怎么知道它会这样”，而是会想“刚才是不是应该卖血打输出”“前面是不是不该拿那张牌”“我是不是太晚删打击了”。

失败能复盘，游戏就会耐玩。

## 五、状态系统让简单卡牌产生复杂组合

游戏里的状态效果很多，但核心逻辑并不难懂。易伤让目标受到更多攻击伤害，虚弱让攻击伤害降低，力量增加攻击伤害，敏捷增加格挡值，还有荆棘、人工制品、脆弱、无实体等各种状态。

这些状态本身不复杂，但它们和卡牌、遗物、药水结合以后，会产生非常多的组合。

比如力量加多段攻击，收益会被放大；易伤配爆发攻击，可以快速压低敌人血量；敏捷配多张格挡牌，会让防御效率明显提高；能力牌提供长期状态收益，但启动时可能亏节奏。

这说明《杀戮尖塔》的复杂度不是堆在单张卡的文字上，而是藏在卡牌之间的关系里。

新手可以先理解最直接的东西：伤害高、防御多、能量够。玩久了以后，才会慢慢理解过牌、消耗、弃牌、能力牌、状态层数、导出伤害这些更深的东西。它的学习曲线不是强迫你先读懂复杂规则，而是让你在一局局里发现组合。

这也是它“易于上手，难于精通”的核心。

## 六、四个角色其实是四套教学方向

知乎那篇把四个角色都讲了一遍，我觉得角色设计也是《杀戮尖塔》成功的重要部分。四个角色不是简单换皮，而是在教玩家理解不同的资源逻辑。

铁甲战士是最适合新手入门的角色。他血量高，初始遗物能在战斗后回血，攻击和防御都比较直观。玩家很容易理解他的基础目标：打伤害，抗住，活下来。

但铁甲战士并不只是简单粗暴。他有防战思路，比如用“壁垒”保留格挡，用“巩固”翻倍格挡，再用“全身撞击”把格挡转成伤害；也有力量流，通过力量增益和多段攻击打爆发；还有消耗生命换收益的牌，比如“放血”一类卡牌，会让玩家理解血量也是资源。

铁甲战士好的地方是入口清晰，上限不低。新手能用攻击和防御玩，老手会开始研究卖血、消耗、力量、格挡导出这些更深的东西。

静默猎手则更强调过牌、毒和小刀。她不像铁甲战士那样靠高基础数值硬打，而是更依赖牌库循环和连续出牌。像“余像”这种每打出一张牌就获得格挡的能力，或者“凌迟”这种每打出一张牌造成伤害的能力，都在鼓励玩家一回合打出大量牌。

毒流也很有代表性。中毒无视格挡，能持续造成伤害，再配合“催化剂”翻倍层数，就会出现很强的后期爆发。猎手教玩家理解另一种强度：不是每回合直接打多少，而是通过叠层、过牌和持续收益压垮敌人。

故障机器人最复杂，因为它有充能球系统。闪电球输出，冰霜球防御，黑暗球蓄力爆发，等离子球提供能量。它还有专注属性，可以提高球的效果。

这个角色的有趣之处在于“启动”。很多时候，故障机器人不是一上来就强，而是需要先放能力牌、造球、提升专注，等系统运转起来以后才厉害。但启动期间很容易被打，所以它对路线、卡牌和遗物的要求更高。

这也解释了为什么玩家会拿它开玩笑，说它启动慢、强度不稳定。它不是单纯弱，而是它的强度更依赖系统搭好。

观者是后来加入的角色，强度公认很高。她的姿态系统非常暴力：平静姿态离开时获得能量，愤怒姿态造成和受到的伤害都翻倍，神圣姿态造成三倍伤害。再配合保留和预见机制，观者可以比较稳定地规划爆发回合。

观者最有意思的是，她把风险收益做得很直接。进愤怒姿态输出翻倍，但自己受到伤害也翻倍。玩家需要判断什么时候进，什么时候退。会玩以后，通过平静和愤怒反复切换，能打出非常高的效率。

四个角色放在一起看，其实是在让玩家从不同角度理解同一件事：资源怎么转换。铁甲战士把血量、防御和伤害互相转换；猎手把出牌次数、毒和过牌转换成强度；机器人把回合启动和充能球转换成长期收益；观者把姿态风险转换成爆发输出。

这比简单做四个职业要好很多。

## 七、地图让宏观决策和战斗决策连在一起

《杀戮尖塔》的地图由普通战斗、精英、火堆、商店、未知事件、宝箱和 Boss 节点组成。玩家每一层都要选择路线。

路线选择是游戏里很重要的宏观决策。它不像战斗出牌那样一回合一回合算，但会决定整局节奏。

打精英风险高，但能拿遗物，长期收益很大。  
火堆可以回血，也可以升级卡牌，但两者通常只能选一个。  
商店可以购买卡牌、遗物、药水，也可以删牌，但前提是金币够。  
未知事件可能赚，也可能亏。  
普通战斗能拿卡和金币，但会消耗血量。  

所以选路线不是选奖励最多，而是选自己现在承受得起的风险。

这一点我觉得非常重要。很多游戏的地图选择只是“多打一关还是少打一关”，但《杀戮尖塔》的路线选择会和卡组状态强绑定。你第一层输出够不够，决定你敢不敢打精英；你血量健康不健康，决定你火堆能不能升级；你金币够不够，决定商店有没有价值；你是否缺关键牌，决定问号和普通战斗的价值。

地图不是战斗之外的包装，而是在决定你能不能把牌组养起来。

Boss 设计也在做类似的检查。第一层 Boss 会检查玩家的基础输出、防御和处理能力；第二层 Boss 开始要求卡组结构更完整；第三层 Boss 则会针对某些策略做限制。比如时间吞噬者限制出牌数量，觉醒者会惩罚能力牌依赖，甜圈和八体有人工制品防止玩家轻易上减益。最后的心脏更是综合检查卡组强度、输出、防御、续航和出牌规划。

这些 Boss 的价值不只是提高难度，而是防止玩家用单一思路打穿所有内容。

## 八、遗物和药水让构筑方向不断被改写

遗物是《杀戮尖塔》里非常重要的长期增益。很多时候，一个遗物会直接改变玩家后面的拿牌思路。

比如“不休陀螺”在没有手牌时抽一张牌，如果牌组里有大量 0 费牌，就可能支撑连续出牌；“异蛇之眼”让玩家每回合多抽牌，但随机化费用，它适合高费牌多的牌组，却可能让原本依赖 0 费牌的牌组非常难受。

这说明遗物不是单纯加数值，而是会改变卡牌价值。

同一张牌，在不同遗物下可能价值完全不同。一个原本普通的 0 费牌，配合某些遗物可能变成核心；一张高费强牌，在普通牌组里很笨重，但配合异蛇之眼可能变得很赚。遗物让构筑不是线性的，而是经常出现转向。

药水则是一次性资源。它不像遗物那样长期生效，但它的即时价值很高。力量药水、复制药水、仙馔密酒这类药水，都可能在关键战斗里改变结果。

我以前玩的时候经常舍不得用药水，总想着留到后面。但从设计角度看，药水的意义正是让玩家拥有一次临时破局能力。它可以帮你打过精英，可以帮你少掉血，可以帮你打赢 Boss。药水槽有限，所以药水也不是越攒越好。你不用，后面可能遇到新药水也拿不了。

这又回到《杀戮尖塔》最核心的设计：资源必须流动起来。太贪、太保守、太想留到以后，都会付出代价。

## 九、进阶和每日挑战延长了游戏生命周期

游戏的生命周期也设计得很完整。角色有解锁进度，会逐步开放新卡牌和遗物。这个设计让新玩家不会一开始就被全部内容淹没，而是边玩边接触新东西。

进阶系统则给老玩家提供了长期目标。20 个进阶等级会逐步增加难度，比如敌人更强、事件更不利、初始诅咒、Boss 连战等。进阶不是简单加血加伤害，而是在压缩玩家的容错率。

低难度下，玩家可以凭直觉和一点运气通关；高进阶下，很多以前能接受的小失误都会被放大。拿牌、路线、药水、火堆、删牌，全部都要更谨慎。

这让《杀戮尖塔》有两层体验：新手能玩懂，高手有得练。它不是一款通关一次就结束的游戏，而是可以一直提高稳定性的游戏。

每日挑战也很有意思。固定随机种子和特殊修改器，让玩家可以在同一条件下比较分数和打法。它把个人游戏变成了社区话题：同一天的规则下，别人怎么打？为什么他能打更高分？

这种设计会自然推动社区讨论。

## 十、社区让游戏的寿命继续延长

知乎那篇提到社区生态，我觉得这也是《杀戮尖塔》成功的一部分。

一个系统足够深的游戏，玩家就会愿意讨论。卡牌强度、遗物价值、路线规划、角色强弱、Boss 应对、流派构筑，这些东西都能形成长期话题。玩家会发攻略，会争论某张牌是不是好牌，会讨论“能力机”“物理机”这样的流派黑话，也会用梗来表达角色强弱和游戏体验。

这种社区讨论反过来又会帮助新玩家入门。一个新玩家如果只靠自己摸索，也能玩，但会比较慢；如果看社区讨论，就能更快理解为什么有些牌表面普通但很强，为什么删牌重要，为什么药水不能乱省。

Mod 生态也延长了游戏寿命。新角色、新卡牌、新遗物、大型机制改造，都能让玩家在熟悉原版之后继续体验新的构筑空间。

我觉得这说明一款策略游戏如果底层规则足够稳，它就很适合长出社区内容。因为玩家不仅是在消费内容，还会围绕系统继续生产理解。

## 十一、它为什么影响了后来的很多游戏

《杀戮尖塔》之后，很多游戏都开始做随机构筑、卡牌战斗和 Roguelike 结合，比如《怪物火车》《月圆之夜》《Inscryption》等。它确实建立了一个很清晰的类型范式。

但我觉得后来的游戏如果只学表面，很容易学成“随机拿卡 + 打怪 + 遗物”。这只是外形，不是核心。

《杀戮尖塔》真正值得学的是这些东西：

第一，随机性必须服务于决策，而不是替代设计。  
第二，玩家失败以后要能复盘，而不是只觉得倒霉。  
第三，奖励要有成本，强策略要有环境限制。  
第四，系统之间要互相影响，而不是平铺。  
第五，入口要简单，但组合上限要高。  
第六，游戏要持续检查玩家短板，避免一个套路通吃。

这些才是它影响后来者的地方。

## 十二、我自己的总结

如果只用一句话概括，我觉得《杀戮尖塔》的伟大之处在于：它把复杂度藏在组合和取舍里，而不是堆在规则说明里。

玩家第一次玩，很快就知道自己要做什么：选路线、打牌、防御、攻击、拿奖励。表层流程非常清楚。

但越玩越会发现，每一个小选择都牵连到后面。第一层拿的一张牌，会影响第二层卡组方向；一次火堆升级，会影响 Boss 战能不能活下来；一个遗物，会改变后续几十张牌的价值；一次贪精英，可能让后面整局都在低血线挣扎。

它最厉害的地方不是让玩家永远拿到爽的东西，而是让玩家永远缺一点东西。缺血、缺能量、缺过牌、缺删牌、缺启动时间、缺输出、缺防御。因为缺，所以必须取舍；因为取舍，所以每个选择都有重量。

读完这三篇以后，我觉得如果以后我自己做游戏，不能只想着加系统、加随机、加卡牌。更应该先问几个问题：

玩家的每次选择到底在交换什么？  
游戏里有没有一个核心资源能把系统串起来？  
随机发生以后，玩家还有没有补救空间？  
失败以后，玩家能不能知道自己哪里做错了？  
强力组合是不是需要条件，而不是随便就能打出来？  
系统有没有持续检查玩家的短板？

《杀戮尖塔》之所以成功，不是因为它有很多内容，而是因为它的内容都围绕决策展开。它让玩家每一局都面对新的问题，又让玩家觉得这些问题是可以通过经验变得更会处理的。

这可能就是它最强的地方：每一次失败都不像终点，更像下一次尝试的理由。
